# Operators
Tools at your hands

## What are Operators?
An operator, in Blender, is a script you can recall no matter the panel. This means that if you make an operator, for example to open a box that says "Hello World!", if the CONTEXT (basically the window) allows it, it will run.
## Are there any in DAT?
Most of the custom scripts I've wrote for DAT are operators. You can use the Blender search bar to find DATools everywhere, no matter the panel you're in. This way you can build your own UI, add DATools to other add-on's panels, or run them with favorites.
