# Mirror It!

This page is a placeholder for a Mirror It! operator.

## Status

The current DATools package does not contain an implemented `dat.mirror_it` operator in the `operators` folder.

## Expected behavior

A Mirror It! operator would typically:
- mirror selected mesh objects across one or more axes,
- use the active object or a chosen axis as the mirror reference,
- preserve object transforms and optionally edit the mesh locally.

## Notes

- Because the operator implementation is not present, there are no actual variables or usage steps available yet.
- If a future version adds `dat.mirror_it`, it should include axis selection and a mirror reference.
