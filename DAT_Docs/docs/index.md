# Welcome to DAT Documentation

This site documents DATools, a Blender add-on that provides workflow utilities for DungeonAlchemist asset preparation.

## What you will find here

- Installation instructions for enabling DATools in Blender.
- A complete operator reference for `Floor It!`, `Scale It!`, `Mirror It!`, `Shrink It!`, `Rez It!`, and `Map It!`.
- A guide to the DATools sidebar, including the Tools, Texture, Light, Script, and Settings panels.
- Information about custom scripts, language support, and current add-on metadata.

## Getting started

1. Install and enable DATools through Blender preferences.
2. Open the 3D View sidebar and select the `DAT` tab.
3. Use the panel selector to open `Tools`, `Texture`, `Light`, `Script`, or `Settings`.
4. Select the objects or materials required by the operator you want to run.

> DATools is made to help the community and is not officially connected with DungeonAlchemist.
